TC349 Week 2: Scalable Navigation

This page demonstrates two separate versions of the navigation exercise found in Dan Cederholm's Bulletproof Web Design (3rd Edition). I have expanded the model outlined in the book by adding a jQuery-powered stylesheet switcher. The two versions share quite a bit of common CSS code, so I've stored that in a main stylesheet, style.css which I included on all pages. All unique bits of code have been stored in two separate css files, image-nav.css and gradient-nav.css for the image and css-only versions respectively. A third stylesheet, both-nav.css, was created to allow for side-by-side comparison between the two versions. Although no changes can be observed when switching stylesheets, proof that the navigation has been re-rendered using different code can be found by inspecting this page using a developer's console such as Firebug.

I also took the liberty of setting up the navigation tabs to actually switch focus when clicked, also using jQuery. In order to accomplish this without leaving the page, I have prevented the default action of the links contained in each tab. The links on the static copies below will work as expected.
